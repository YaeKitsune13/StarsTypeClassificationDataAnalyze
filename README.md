# StarsTypeClassificationDataAnalyze

Домашняя работа по классификации типов звёзд и анализу астрономических данных в формате Jupyter Notebook.  
В репозитории содержатся ноутбуки с:

- предварительной обработкой и исследовательским анализом данных о звёздах;
- построением и обучением моделей машинного обучения для предсказания типа звезды;
- сравнением качества различных моделей и подбором гиперпараметров;
- визуализацией результатов классификации и характеристик выборки.
